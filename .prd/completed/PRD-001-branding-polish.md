# PRD: Pocket Marketer Branding Polish

**ID:** PRD-PM-001  
**Scope:** M (20 min)  
**Priority:** P1

## Context
Pocket Marketer has a dark navy theme started. Need to polish branding for professional appearance.

## Objective
Complete branding polish: logo placement, favicon, loading states, and consistent visual identity.

## PM Brand Guidelines
- **Primary Background:** #0a1628 (dark navy)
- **Secondary:** #0d1f35, #132743
- **Accent:** #3b82f6 (blue)
- **Text:** #ffffff (primary), #94a3b8 (secondary)

## Tasks
1. **Logo Integration:**
   - Ensure logo-dark-styled.png is used in header
   - Add logo to loading states
   - Update favicon to PM brand

2. **Header Polish:**
   - Clean PM branding in sidebar
   - Remove any bolt.diy references
   - Add "Pocket Marketer" text next to logo

3. **Loading States:**
   - Add PM logo to initial loader
   - Brand the "Generating..." states
   - Consistent loading animations

4. **Footer/Credits:**
   - Add subtle "Powered by Pocket Marketer" footer
   - Remove any bolt.new references

## Acceptance Criteria
- [ ] PM logo visible in header
- [ ] No bolt.diy/bolt.new branding visible
- [ ] Loading states show PM branding
- [ ] Favicon updated
- [ ] npm run build passes
